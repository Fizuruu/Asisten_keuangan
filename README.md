# 💸 Asisten Keuangan ✨
### Financial Assistant - Anime Edition

Aplikasi pencatat keuangan harian Android yang modern, profesional, dan estetik dengan tema **Anime Red**.

---

## 📸 Tampilan Utama

- 🎨 Tema: Anime Kawaii (Glassmorphism Style)
- ❤️ Warna Dominan: `#FF4D4D`
- 🖼️ Background: `background.jpg`

---

## 🚀 Fitur Unggulan

- 💰 Dashboard Saldo Real-time
- 💸 Pemisahan Tab Transaksi
- ✏️ Tambah, Edit, Hapus transaksi
- 😀 Kategori berbasis emoji
- 📊 Grafik Pie Chart interaktif
- 💾 Penyimpanan lokal Room Database
- 👉 Swipe to Action

---

## 🛠️ Tech Stack

| Teknologi | Keterangan |
|---|---|
| Java | Bahasa utama |
| MVVM | Arsitektur |
| Room | Database |
| Material Design 3 | UI |
| MPAndroidChart | Grafik |

---

---

## ⚙️ Cara Menjalankan

1. Clone project
2. Buka di Android Studio
3. Tambahkan `background.jpg`
4. Sync Gradle
5. Run aplikasi
